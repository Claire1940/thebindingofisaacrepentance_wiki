<Section icon={Settings} title="Instalando o Mod External Item Descriptions para The Binding of Isaac Repentance">

Instalar o mod **the binding of isaac repentance external item descriptions** (EID) é un processo simples através da Oficina Steam (Steam Workshop). Este mod funciona como uma sobreposição em tempo real, lendo a memória interna do jogo para fornecer dados precisos sem desativar conquistas (desde que você tenha derrotado a Mom pelo menos uma vez). Para jogadores que buscam a conquista Dead God, o EID é indispensável para identificar quais itens contribuem para transformações específicas ou possuem anti-sinergias perigosas.

<Callout type="success" title="Segurança de Conquistas">
Em Repentance, os mods não desativam conquistas ou marcas de conclusão, desde que você tenha derrotado a chefe Mom em Depths pelo menos uma vez nesse arquivo de salvamento.
</Callout>

<Steps>
  <Step num="1" title="Acesse a Oficina Steam">
    Abra sua Biblioteca Steam, selecione The Binding of Isaac: Rebirth e clique na aba Oficina (Workshop). Certifique-se de ter a DLC Repentance instalada, pois as versões modernas do EID são otimizadas para esta versão.
  </Step>
  <Step num="2" title="Pesquise e Inscreva-se">
    Pesquise por "External Item Descriptions" do autor Wayu. Clique no botão verde Inscrever-se. A Steam fará o download automático dos arquivos necessários para o seu diretório local.
  </Step>
  <Step num="3" title="Ative no Menu do Jogo">
    Inicie o jogo e navegue até o menu Mods a partir da tela de título. Certifique-se de que "External Item Descriptions" esteja destacado e ativado. Se o texto estiver cinza, pressione a tecla Tab para ativar os mods globalmente.
  </Step>
  <Step num="4" title="Instale o Mod Config Menu">
    Para a melhor experiência, inscreva-se também em "Mod Config Menu - Pure" na Oficina. Isso permite que você altere as configurações do EID, como idioma ou tamanho da fonte, enquanto o jogo está rodando.
  </Step>
</Steps>

<Section icon={Info} title="Recursos Principais e Mecânicas de Exibição">

A função principal do mod **the binding of isaac repentance external item descriptions** é eliminar a necessidade de buscas externas em wikis. Quando Isaac fica perto de um item de pedestal, uma caixa de texto aparece no lado esquerdo da tela. Esta caixa contém o nome do item, sua qualidade (ranqueada de 0 a 4) e um detalhamento detalhado de seus efeitos mecânicos.

<Callout type="tip" title="Qualidade Importa">
Os itens são ranqueados da Qualidade 0 (fraco/situacional) à Qualidade 4 (define a partida). O EID exibe esse número com destaque, ajudando você a decidir se deve pegar um item ou usar uma ferramenta de rerrolagem como o D6.
</Callout>

Além de pedestais simples, o EID rastreia vários outros elementos críticos de jogabilidade:
- **Berloques (Trinkets):** Exibe o efeito passivo do berloque segurado ou caído no chão.
- **Cartas e Runas:** Explica exatamente o que uma carta faz antes de você usá-la, incluindo cartas de tarô invertidas.
- **Pílulas:** Uma vez que uma pílula é identificada, o EID mostra seu efeito. Também pode mostrar efeitos para pílulas não identificadas se você tiver os itens "PHD" ou "False PHD".
- **Transformações:** Mostra um pequeno ícone indicando se um item contribui para transformações como Guppy, Beelzebub ou Conjoined.

| Elemento de Exibição | Descrição | Indicador Visual |
| :--- | :--- | :--- |
| **Nome do Item** | O nome oficial do coletável | Cabeçalho em Negrito |
| **Texto de Efeito** | Mudanças de atributos (Dano, Lágrimas, etc.) | Lista com Marcadores |
| **Qualidade** | Nível de poder de 0 a 4 | Número Colorido |
| **Pool de Itens** | De onde o item se originou (Anjo, Demônio, etc.) | Rótulo de Texto |
| **Origem do Mod** | Indica se o item é de um mod | Subtexto |

</Section>

<Section icon={Target} title="Funcionalidade Avançada para Personagens Tainted">

Para jogadores avançados, o mod **the binding of isaac repentance external item descriptions** fornece ferramentas especializadas para personagens complexos. Tainted Cain, por exemplo, depende inteiramente do Bag of Crafting. O EID lê os consumíveis (moedas, bombas, chaves, corações) atualmente no chão da sala e gera uma lista de itens possíveis que você pode criar.

<Callout type="warning" title="Receitas do Bag of Crafting">
Na versão atual de Repentance, muitas receitas do Bag of Crafting são aleatorizadas com base na semente (seed) da sua partida. O EID calcula isso localmente, garantindo que as receitas exibidas sejam precisas para o seu jogo específico.
</Callout>

Outra interação poderosa é com o **Spindown Dice**. Quando você segura o Spindown Dice, o EID exibirá em qual item o pedestal atual se transformará após um uso. Isso é vital para buscar itens de alto valor, como Death Certificate ou Sacred Heart, ao "girar para baixo" itens de nível inferior com números de ID específicos.

<Section icon={Shield} title="Configuração e Otimização">

Para obter o máximo do mod **the binding of isaac repentance external item descriptions**, você deve personalizar a exibição para se adequar à sua resolução. Ao usar as configurações padrão, o texto pode se sobrepor aos recursos "Found HUD" ou "Extra HUD" integrados ao jogo.

| Categoria de Configuração | Valor Recomendado | Impacto na Jogabilidade |
| :--- | :--- | :--- |
| **Idioma** | English (ou Nativo) | Altera as descrições para o seu idioma de preferência |
| **Escala do Texto** | 0.8x a 1.0x | Evita que a interface polua a tela |
| **Modo de Exibição** | Detalhado | Mostra mudanças exatas de atributos numéricos |
| **Transparência** | 0.5 | Permite que você veja inimigos atrás da caixa de texto |

<Callout type="info" title="Alinhamento do HUD">
Se você usa uma proporção de tela 4:3 ou joga no Steam Deck, use the Mod Config Menu para mover a janela do EID para o lado direito ou para o centro para evitar bloquear sua barra de vida.
</Callout>

O mod também suporta "Ícones Indicadores" para tipos de salas especiais. Por exemplo, em uma Sala de Sacrifício (Sacrifice Room), o EID pode mostrar as recompensas para cada batida subsequente nos espinhos, como a chance do 6º golpe teleportar para a Sala de Anjo ou a chance do 7º golpe conceder um item de Anjo.

</Section>
