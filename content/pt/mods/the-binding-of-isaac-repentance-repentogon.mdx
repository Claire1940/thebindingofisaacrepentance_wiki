<Section icon={Map} title="Como Instalar o Repentogon">

O processo de instalação é simples, mas requer a colocação manual dos arquivos, já que injetores de código nativo não podem ser distribuídos diretamente através do Steam Workshop padrão. Siga estes passos cuidadosamente para garantir uma configuração limpa sem arriscar o progresso do seu save atual.

<Steps>
  <Step num="1" title="Fazer Backup dos Dados de Salvamento">
    Localize sua pasta de salvamento, normalmente encontrada no diretório de dados do usuário da Steam ou na pasta Meus Jogos (My Games) no Windows. Copie a pasta inteira para um local de backup seguro na sua área de trabalho.
  </Step>
  <Step num="2" title="Baixar o Pacote de Lançamento">
    Visite o repositório oficial do GitHub para o projeto e baixe o arquivo zip compilado mais recente. Certifique-se de baixar apenas arquivos de fontes oficiais verificadas para evitar riscos de segurança.
  </Step>
  <Step num="3" title="Extrair Arquivos para o Diretório Raiz">
    Abra sua biblioteca da Steam, clique com o botão direito no jogo, selecione Gerenciar e clique em Explorar Arquivos Locais. Extraia o conteúdo do zip baixado diretamente nesta pasta raiz, ao lado do executável principal.
  </Step>
  <Step num="4" title="Verificar a Instalação">
    Inicie o jogo através da Steam. Olhe para o menu principal; uma string de versão personalizada ou um logotipo sobreposto deve aparecer no canto, confirmando que o hook nativo foi carregado com sucesso.
  </Step>
</Steps>

<Callout type="success" title="Instalação Verificada">
Assim que o logotipo personalizado aparecer no menu principal, o framework estará ativo. Você pode agora se inscrever com segurança em mods avançados do Steam Workshop que exigem este extensor.
</Callout>

</Section>

<Section icon={Target} title="Solução de Problemas Comuns">

Se o seu jogo não iniciar ou travar frequentemente após a instalação, a causa geralmente é um erro simples de configuração ou um conflito com sobreposições de terceiros desatualizadas. A maioria dos problemas pode ser resolvida verificando os arquivos do jogo ou atualizando as bibliotecas de tempo de execução (runtime) no seu sistema operacional.

Se você tiver problemas persistentes no Linux ou Steam Deck, verifique novamente as opções de inicialização do Proton. O comando de substituição de DLL deve ser escrito exatamente como especificado na documentação oficial para force o ambiente Wine a carregar a biblioteca injetada.
