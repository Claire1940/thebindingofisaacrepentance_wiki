Ao ter um amigo jogando com um personagem mais resistente, você pode proteger "canhões de vidro" frágeis enquanto ainda colhe as recompensas de uma partida bem-sucedida.

<Section icon={Map} title="Como Configurar Sessões de Co-op Local">

Configurar uma sessão para o **co-op local de The Binding of Isaac: Repentance** é simples, mas requer configurações específicas de hardware para garantir que o jogo reconheça várias entradas. Embora a versão para PC seja a mais flexível, as versões para console (Switch, PS5, Xbox) oferecem suporte nativo plug-and-play para controles adicionais.

<Steps>
  <Step num="1" title="Conectar Controles">
    Certifique-se de que todos os gamepads estejam conectados ao seu PC ou console antes de iniciar o jogo. No PC, o Steam Input geralmente cuida do mapeamento, mas ferramentas de terceiros podem ser necessárias para controles mais antigos.
  </Step>
  <Step num="2" title="Iniciar uma Nova Partida">
    O Jogador 1 inicia uma partida padrão (modo Normal, Hard ou Greed). Você deve estar na primeiríssima sala (Sala Inicial) para adicionar jogadores adicionais.
  </Step>
  <Step num="3" title="Entrar no Jogo">
    Os jogadores adicionais devem pressionar o botão 'Start' ou 'Options' em seus controles. Um menu de seleção de personagens aparecerá para cada novo jogador.
  </Step>
  <Step num="4" title="Selecionar Personagens">
    Cada jogador escolhe seu personagem. Note que não é possível ter personagens duplicados (por exemplo, dois Isaacs), a menos que use mods ou glitches específicos.
  </Step>
</Steps>

<Callout type="warning" title="Limitação de Hardware">
No PC, o Jogador 1 pode usar o teclado, mas os Jogadores 2, 3 e 4 **devem** usar controles. O jogo geralmente não suporta vários teclados para jogo local sem softwares de remapeamento externos complexos.
</Callout>

</Section>

<Section icon={Shield} title="Dominando as Mecânicas do Co-op Local de The Binding of Isaac: Repentance">

O sucesso no **co-op local de The Binding of Isaac: Repentance** depende da distribuição de recursos. Como o jogo não dimensiona a quantidade de itens dropados com base no número de jogadores, uma equipe de quatro jogadores terá a mesma quantidade de itens que um jogador solo. Isso cria um nível de poder "diluído", onde cada jogador é individualmente mais fraco do que um Isaac solo seria.

Para compensar, os jogadores devem utilizar as forças específicas de cada personagem. Por exemplo, dar todos os itens de aumento de vida (HP Up) para Magdalene permite que ela atue como um tanque para a equipe, enquanto dar multiplicadores de dano para Azazel garante que a equipe tenha DPS suficiente para derrotar os chefes rapidamente.

| Tipo de Recurso | Estratégia de Gerenciamento | Alvo Prioritário |
| :--- | :--- | :--- |
| **Corações Vermelhos** | Deixe no chão para Magdalene/Samson | **Tanques / Personagens que se Automutilam** |
| **Corações de Alma** | Proteja as chances de Pacto para toda a equipe | **Personagens Frágeis** |
| **Itens Passivos** | Atribua com base nas sinergias existentes | **Causador de Dano Principal** |
| **Itens Ativos** | Diversifique (1 de Cura, 1 de Dano, 1 de Utilidade) | **Personagens de Suporte** |
| **Ornamentos (Trinkets)** | Troque frequentemente para maximizar os benefícios do andar | **Personagens Baseados em Sorte** |

<Callout type="success" title="Bônus de Sinergia">
Itens que afetam a "sala inteira" ou "todos os inimigos" tornam-se significativamente mais valiosos no co-op. Itens como **The Relic** ou **Book of Revelations** fornecem vida que pode ser distribuída para quem mais precisa, mantendo toda a equipe viva.
</Callout>

</Section>
