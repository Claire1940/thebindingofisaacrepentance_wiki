<Section icon={Swords} title="Shared Progression and Co-op Mechanics">

The multiplayer system introduces unique rules for resource distribution and progression tracking. Understanding how the game handles pickups, item pedestals, and completion marks is essential for planning successful deep runs.

While players control individual characters with separate health bars and active items, the utility resources are completely shared. Coins, bombs, and keys go into a single unified counter visible at the top of the screen. Any player can spend these resources, requiring active communication before opening locked chests, purchasing shop items, or using bombs to search for secret rooms.

| Resource Type | Distribution | Strategy |
| --- | --- | --- |
| Coins | Shared Pool | Save for shops, priority items, or key health purchases |
| Bombs | Shared Pool | Prioritize tinted rocks, secret rooms, and boss damage |
| Keys | Shared Pool | Always reserve at least one key for the next Treasure Room |
| Health | Individual | Red, Soul, and Black hearts are picked up per character |
| Pedestal Items | Single Pickup | Assign items to characters based on synergy value |

<Callout type="tip" title="Resource Coordination">
Because keys and bombs are shared, players must coordinate their spending. Avoid wasting bombs on basic obstacles if a future floor requires key access to a Planetarium or a locked vault.
</Callout>

Progression marks earned during an online co-op run apply to all eligible players in the lobby. If the group successfully defeats a major boss, such as Mother or The Beast, every player who survived the run receives the corresponding completion mark on their active save file, provided they selected a character that still needed that specific victory mark.

| Boss Route | Unlock Reward Type | Target Priority |
| --- | --- | --- |
| Mom's Heart | Core Progression | Essential for unlocking alternative chapters |
| Cathedral / Sheol | Chapter V Marks | Unlocks powerful character-specific passive items |
| Chest / Dark Room | Endgame Items | Provides high-impact pool upgrades like the Polaroid |
| Mega Satan | Golden Gate | Unlocks Apollyon and community-favorite trinkets |
| The Beast | Tainted Variants | Unlocks the ultimate endgame character roster |

This shared progression makes online co-op an excellent tool for helping friends complete difficult character post-it notes, such as those for Jacob and Esau or Tainted Lost. A skilled player can carry the run using a strong, reliable character while the other player focuses on survival to secure the completion mark.

</Section>
