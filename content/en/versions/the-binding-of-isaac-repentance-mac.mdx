<Section icon={Target} title="Step-by-Step Mac Installation Guide">

Setting up the game via Porting Kit is the most reliable free method for both Intel and Apple Silicon Macs. This tool automatically configures a custom Wine wrapper designed specifically to run the Windows version of Steam.

Follow these steps to complete your installation:

<Steps>
  <Step num="1" title="Download and Install Porting Kit">
    Visit the official Porting Kit website and download the latest version for macOS. Drag the application into your Applications folder and launch it. Grant any necessary system permissions during the first startup.
  </Step>
  <Step num="2" title="Install the Windows Steam Wrapper">
    Inside Porting Kit, navigate to the **Library** tab and search for **Steam**. Select the Windows Steam option and click **Install**. The program will automatically download Wine dependencies and create a dedicated Windows container on your drive.
  </Step>
  <Step num="3" title="Log In and Download the Game">
    Once the installation finishes, launch Steam through Porting Kit. Log into your Steam account, find *The Binding of Isaac: Rebirth* in your library, and make sure the *Repentance* DLC is checked in the DLC management panel. Click install to download the Windows files.
  </Step>
  <Step num="4" title="Configure Controller and Launch">
    Connect your controller via Bluetooth or USB. In Steam's controller settings, enable **Steam Input** for your specific gamepad (Xbox, PlayStation, or Switch Pro). Launch the game to confirm that inputs and save data are syncing correctly.
  </Step>
</Steps>

<Callout type="success" title="Steam Cloud Sync">
Because you are running the official Windows version of Steam inside the wrapper, your cloud saves will sync perfectly. If you previously played Repentance on a Windows PC, your progress, characters, and unlocks will carry over immediately.
